Альфа-Банк Хакатон: AI-Помощник для малого бизнеса
Веб-приложение-чат-бот для помощи владельцам малого бизнеса с использованием AI. Проект создан для хакатона Альфа-Банка.
**Запуск одной командой**: `docker-compose up -d --build` - все уже настроено!



## Используемая нейросеть
**OpenRouter API** с бесплатными моделями:

- **Основная модель**: `mistralai/mistral-7b-instruct:free` - Mistral 7B Instruct (free)
  - Стабильная, качественная модель
  - Полностью бесплатная
  - Работает без VPN
- **Альтернативная модель**: `google/gemini-2.0-flash-exp:free` - Gemini 2.0 Flash Experimental
  - Очень быстрая модель (до 161 tps)
  - Отличное качество ответов
  - Полностью бесплатная
- **Fallback модель**: `meta-llama/llama-3.2-3b-instruct:free` - Llama 3.2 3B Instruct
  - Используется если основные модели недоступны
  - Может быть временно rate-limited
- **Все модели полностью бесплатные** (max_price=0) и работают без VPN.
  - Получить API ключ: https://openrouter.ai/keys



## Запуск

### Вариант 1: Docker
1. **Клонируйте репозиторий**:
```bash
git clone <repository-url>
cd ALFA
```
2. **Запустите проект** (API ключ должен быть в файле `backend/.env`):
```bash
docker-compose up -d --build
```
Или если используете новую версию Docker:
```bash
docker compose up -d --build
```
3. **Откройте в браузере**:
- Frontend: http://localhost:3000
- Backend API: http://localhost:8080

### Вариант 2: Локальный запуск
#### Backend (Golang)
1. **Установите Go** (версия 1.21 или выше)
2. **Перейдите в директорию backend**:
```bash
cd backend
```
3. **Установите зависимости**:
```bash
go mod tidy
```
4. **API ключ уже настроен** в файле `.env` в корне проекта
5. **Запустите сервер**:
```bash
go run main.go
```
Backend будет доступен на `http://localhost:8080`

#### Frontend (Next.js)
1. **Установите Node.js** (версия 18 или выше)
2. **Перейдите в директорию frontend**:
```bash
cd frontend
```
3. **Установите зависимости**:
```bash
npm install
```
4. **Настройки уже готовы** - API URL настроен автоматически
5. **Запустите dev сервер**:
```bash
npm run dev
```
Frontend будет доступен на `http://localhost:3000`



## Технологии

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Golang 1.21+, Gin framework
- **Database**: SQLite
- **AI**: OpenRouter API (Mistral 7B, Gemini 2.0 Flash, Llama 3.2)
- **Authentication**: JWT токены
- **File Processing**: Поддержка Word (.docx) и Excel (.xlsx) файлов



## Функционал

### Основные возможности:
- **Регистрация и авторизация** с указанием:
  - Имя пользователя (username)
  - Название бизнеса
  - Специализация бизнеса
- **Загрузка файлов** с данными о бизнесе:
  - Текстовые файлы (.txt, .csv)
  - Word документы (.docx)
  - Excel таблицы (.xlsx)
- **AI-чат-бот** с категориями вопросов:
  - Финансовый анализ
  - Юридические вопросы
  - Вопросы по персоналу
  - Маркетинг и продвижение
  - Рост и развитие
  - Отчеты и аналитика
  - Общие вопросы
- **Анализ загруженных файлов** - AI анализирует содержимое файлов при ответе
- **История сообщений** - сохранение всех чатов
- **Темная/светлая тема** - переключение темы оформления
- **Адаптивный дизайн** - работает на мобильных устройствах

### Особенности AI:
- AI знает владельца бизнеса (username)
- AI знает название бизнеса
- AI знает специализацию бизнеса
- AI анализирует загруженные файлы
- AI дает конкретные, практические советы
- AI учитывает специфику малого бизнеса

## Использование
1. **Зарегистрируйтесь**:
   - Укажите имя пользователя, пароль
   - Укажите название вашего бизнеса
   - Укажите специализацию бизнеса
2. **Загрузите файлы**:
   - Перейдите во вкладку "Файлы"
   - Загрузите документы о вашем бизнесе (отчеты, данные о сотрудниках, финансовые документы)
   - Поддерживаются: .txt, .csv, .docx, .xlsx
3. **Начните чат**:
   - Выберите категорию вопроса или задайте общий вопрос
   - AI проанализирует ваши файлы и даст персональный ответ
   - Все чаты сохраняются в истории
4. **Управляйте чатами**:
   - Создавайте новые чаты для разных тем
   - Просматривайте историю сообщений
   - Удаляйте ненужные чаты



## Безопасность
- Пароли хешируются с использованием bcrypt
- JWT токены для аутентификации
- CORS настроен для безопасности
- Валидация входных данных
- Защита от SQL инъекций (параметризованные запросы)



## База данных
Используется SQLite с автоматическими миграциями:
- Таблица `users` - пользователи
- Таблица `files` - загруженные файлы
- Таблица `chats` - чаты
- Таблица `messages` - сообщения
База данных создается автоматически при первом запуске в директории `database/alfa_hack.db`

## Стоит учесть!!!
При запуске проекта если не работает нейросеть - то может быть закрылся API-ключ, пожалуйста, сообщите если это так и есть
- OpenRouter API-ключ иногда автоматически временно деактивируется (например, после множества тестовых запросов подряд, перезапуска или в случае необычной нагрузки).
- Если искусственный интеллект перестал отвечать — зайдите на https://openrouter.ai/keys, создайте новый ключ, вставьте его в backend/.env (заменяя старый)
